UI Elements (components)

Best practices for your next digital prototype...

Here you’ll find a UI and design library for folks at Dell and beyond, so you can complete projects even faster. By using our framework and components, you’re helping us ensure consistency and create a cohesive Dell customer experience everywhere on the web and on every device.

NOTE: When using the components from the Dell Design Library if you keep the component inside a div tag with a container class...you will basically make the component a building block. This in return will make building the pages faster and more consistant.

Example

   <div class="container">
     <div class="row">
       <div class="col-md-9">[ Component goes here ] </div>
       <div class="col-md-3">[ This is setup for nav within a page ] </div>
     </div>
   </div>
