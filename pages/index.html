Things to know

General frame layout of page.

  • Masthead
  • Body Content
  • Footer

Layer structure within the body content is:

  • Section
  • Container - There are two types of containers
    • Container
    • Container-fluid
  • Row
  • Columns - There are four different size columns. Here is the order.
    • xs - extra small (.col-xs-#)
    • sm - small (.col-sm-#)
    • md – medium ( .col-md )
    • lg – large ( .col-lg )
  1. Rows must be placed within a .container (fixed-width) or .container-fluid (full-width) for proper alignment and padding.
  2. Use rows to create horizontal groups of columns.
  3. All rows contain columns and each row contains 12 columns.
  4. Content should be placed within columns, and only columns may be immediate children of rows.
